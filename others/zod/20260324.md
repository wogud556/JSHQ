### zod
- zod는 타입스크립트를 위한 스키마 기반의 데이터 검증 라이브러리
- 컴파일 타임에서 타입을 검사하는 Typescript와 달리 런타임에서도 데이터의 유효성을 검사할 수 있도록 도와줌

### zod의 특징
- 런타임 타입 검증
  - 데이터를 실행중에 검증 가능
- 타입스크립트 친화적
  - 스키마에서 TypeScript 타입을 자동 생성
- 체이닝 방식의 API
  - 직관적이고 간결한 API 제공
- 강력한 유효성 검사
  - 다양한 검증 메소드 제공 (min(), max(), email())
- 구조적 오류 메시지
  - 에러 발생 시 상세한 정보를 제공

### 기본 사용법
- import { z } from 'zod'를 import 하고 사용함

### 스키마 정의
- zod에서 지원하는 타입 예시
  - z.string(), z.number() 등 다양한 타입을 제공함
```
import { z } from 'zod';

// primitive values
z.string();
z.number();
z.bigint();
z.boolean();
z.date();
z.symbol();

z.function();

z.literal('hello')
z.array(z.number());

z.undefined();

z.null();
z.void();

z.any();
z.unknown();

z.never();

z.object({
    str: z.string(),
    num: z.number(),
    bool: z.boolean(),
    date: z.date(),
    arr: z.array(z.number()),

})

```

### 숫자 스키마 정의
- 숫자는 z.number()로 선언함
- 기본적으로 Javascript의 number 타입을 따르므로 Number.MIN_VALUE, Number.MAX_VALUE 사이의 값을 가짐
- z.number()와 z.bigInt()는 서로 다른 타입을 검증하므로, bigint를 사용할 때는 반드시 n을 붙여야 함
- bigint는 number와 다르게 소수점을 지원하지 않음
