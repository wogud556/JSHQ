## 타입스크립트 기초
- 변수의 타입이 어떤건지 알 방법이 필요함
- 타입스크립트를 사용하면 아래와 같이 작성이 가능함
```
function getUserByID(userID : number) {
    // 사용자 id를 이용하여 사용자 정보 조회 api를 호출함
}
```
- 완벽하다 이 코드에서는 자바스크립트와는 달리 인수가 숫자 자료형인 것을 알 수 있고, 문자열을 전달하면 오류가 발생함
- 간단히 자료형을 명시하는 것만으로도 코드가 명확해짐


### 타입스크립트란?
- 자료형이 있는 자바스크립트의 상위 집합이라고 볼 수 있음
- 상위집합이라는 뜻은 타입스크립트 파일에 일반 자바스크립트를 작성해도 되며 오류가 발생하지 않음을 의미함
- 브라우저는 타입스크립트를 이해하지 못하기 때문에 일반 자바스크립트를 트랜스파일 해야함

### 타입스크립트 사용 방법
- 타입스크립트를 시작하는데 5분이면 충분
- 아래처럼 typescript를 설치함
```
npm install -g typescript
```
- 다음으로 코드 편집기를 열고 gretter.ts(.js아님)라는 파일을 만들어본다

```
const gretter = (name : string) => {
    console.log(`hello ${name}`);
}
greeter('Alberto');
//hello alberto


```

### 기본 자료형
- boolean
- number
- string
- Array
- object
- 튜플
- enum
- any
- void
- null 과 undefined
- never
- 하나씩 간단히 살펴본다

### boolean
- true 또는 false가 될 수 있는 값을 정의함
```
const active : boolean = true;
```

### number
- number 자료형은 자바스크립트와 마찬가지로 10진 16진 2진, 8진 리터럴을 지원함

```
const decimal: number = 9;
const hex : number = 0xf00d;

```
