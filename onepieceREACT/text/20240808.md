## 변수와 상수
- 값을 저장하는 상자

## 자료형

### 원시티압
- Number
- String
- Boolean
- Null
- Undefined

### 객체타입
- Object
  - Array
  - Function
  - RegexExp

### 형변환
- 묵시적 형변환
  - 개발자가 설정하지 않아도 알아서 자바스크립트 엔진이 알아서 형변환
- 명시적 형변환
  - 개발자가 직접 함수 등을 이용해 형변환을 일으킴

### 연산자
- "+ - * /"


### 조건문
- 특정 조건을 만족했을 때에만 실해되는 코드를 작성하기 위한 문법
- 대표적으로 if, switch 조건문이 존재함
